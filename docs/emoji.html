The emoji-only stack is only suitable for wrapping emoji, not text.

Forced emoji and non-emoji refer to the opt-in and opt-out mechanism, called “font variants,” introduced to allow an ambiguous code point to be shown as symbol or emoji (&#xFE0F; and &#xFE0E;, respectively).

The two-men-kissing emoji is actually a composite: &#x1F468;&#x200D;&#x2764;&#xFE0F;&#x200D;&#x1F468;, or “man + joiner + ambiguous heart + emoji font variant selector + joiner + man.”

Similarly, the rainbow / pride flag is “(ambiguous) flag + emoji font variant selector + joiner + rainbow.”
